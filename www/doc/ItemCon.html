Class: ItemCon

ItemCon

new ItemCon(context, lookupName, model, defn)

An Item Context is created for every distinct data_name that is accessed over the lifetime of a given CIF data_block. Used within a Looped Category context, a single ItemCon is recycled (reInit()) for use with each row accessed.
Parameters:
Name Type Description
context Context a reference to the parent data_block's Context.
lookupName string is the string value of the _data.item name, in DDLm form
model ItemCtrlr is a reference to either the data_item or its loop_wrapper.
defn DictItem a reference to the dictionary definition of this _data.item
