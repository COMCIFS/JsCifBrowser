Class: CatCon

CatCon

new CatCon(context, lookupName, model, defn)

CatCon maintains the execution state for categories. Every category that has _data.item children, that are accessed as part of an evaluation, requires a CatCon instance i.e. there is one for every category.
Parameters:
Name Type Description
context a reference to the parent data_block's execution context.
lookupName the name of the category.
model is a reference to the corresponding CIF data category's container.
defn is a reference to the dictionary definition for this category.
